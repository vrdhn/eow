Use this website to develop, share, and refine espeak-ng language files.

On pressing 'Play', the entered data is sent to server, and WAV file is geneated.

With screen reader, this can be a little confusing, hence audio starts and finish with beep.

On pressing 'Share', an unique sharable URL is generated.

Please do not abuse the service.
